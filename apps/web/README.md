# SendAm Web App

This is the Next.js frontend for SendAm.

## Features
- Landing page
- Wallet test interface
- Admin dashboard

## Tech Stack
- Next.js 14 (App Router)
- React
- Tailwind CSS

## Setup
1. Copy `.env.local.example` to `.env.local`
2. Run from the root of the monorepo: `npm install`
3. Run the dev server: `npm run dev:web`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
